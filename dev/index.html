BaseExtensions.allzero — Method
allzero(a)

Check if all elements of a are zero.

Examples

julia> a = [1, 2, 0];

julia> allzero(a)
false

julia> a = zeros(2);

julia> allzero(a)
true

BaseExtensions.flipdims — Method
flipdims(A)

Permute A so that the sequence of axes is reversed. For 2D arrays this is an eager transpose. The operation is not recursive.

Examples

julia> A = reshape(collect(1:8), 2, 4)
2×4 Array{Int64,2}:
 1  3  5  7
 2  4  6  8

julia> B = flipdims(A)
4×2 Array{Int64,2}:
 1  2
 3  4
 5  6
 7  8

julia> A = reshape(collect(1:8), 2, 2, 2)
2×2×2 Array{Int64,3}:
[:, :, 1] =
 1  3
 2  4

[:, :, 2] =
 5  7
 6  8

julia> flipdims(A)
2×2×2 Array{Int64,3}:
[:, :, 1] =
 1  3
 5  7

[:, :, 2] =
 2  4
 6  8

BaseExtensions.reinterpretcomplex — Method
reinterpretcomplex(arr::AbstractArray{<:Real})

Reinterpret a Real array as a Complex one assuming that the real and imaginary parts are laid out consecutively.

Examples

julia> a = [1, 2, 3, 4];

julia> reinterpretcomplex(a)
2-element reinterpret(Complex{Int64}, ::Array{Int64,1}):
 1 + 2im
 3 + 4im

See also: reinterpretreal

BaseExtensions.squeeze — Method
squeeze(A)

Remove dimensions from A that have size(A, dim) == 1. Returns a view that shares the underlying data with A.

Examples

julia> A = reshape(collect(1:4), 1, 4)
1×4 Array{Int64,2}:
 1  2  3  4

julia> B = squeeze(A)
4-element Array{Int64,1}:
 1
 2
 3
 4
